## 1. 전체 구조 개요

### 1.1 레이아웃 구조
```
┌─────────────────────────────────────────────────────────────┐
│                         HEADER                              │
│  [Logo/Title]                    [Lang] [PDF] [Upload]      │
├─────────────────────────────────────────────────────────────┤
│                    UPLOAD SECTION (초기)                     │
│                  또는 DASHBOARD SECTION                      │
│ ┌───────────────────────────────────────────────────────┐  │
│ │                   PERIOD SELECTOR                      │  │
│ │     [전체 기간]  [최근 1달]  [최근 1주]                │  │
│ └───────────────────────────────────────────────────────┘  │
│ ┌───────────────────────────────────────────────────────┐  │
│ │                   STATISTICS CARDS                     │  │
│ │  ┌──────┐ ┌──────┐ ┌──────┐ ┌──────┐ ┌──────┐       │  │
│ │  │Card 1│ │Card 2│ │Card 3│ │Card 4│ │Card 5│       │  │
│ │  └──────┘ └──────┘ └──────┘ └──────┘ └──────┘       │  │
│ └───────────────────────────────────────────────────────┘  │
│ ┌───────────────────────────────────────────────────────┐  │
│ │                    TAB NAVIGATION                      │  │
│ │ [전체현황][TQC분석][불량분석][위험도][업무지시][지속성]│  │
│ └───────────────────────────────────────────────────────┘  │
│ ┌───────────────────────────────────────────────────────┐  │
│ │                    TAB CONTENT                         │  │
│ │  (Active Tab Content Area)                            │  │
│ └───────────────────────────────────────────────────────┘  │
└─────────────────────────────────────────────────────────────┘
```

### 1.2 색상 팔레트
- **Primary**: `#2563eb` (파란색) - 주요 액션, 활성 상태
- **Success**: `#10b981` (녹색) - 양호한 상태, 성공
- **Warning**: `#f59e0b` (주황색) - 경고, 주의 필요
- **Danger**: `#ef4444` (빨간색) - 위험, 불량률 높음
- **Gray Scale**: `#111827` ~ `#f9fafb` - 텍스트, 배경, 테두리

### 1.3 타이포그래피
- **헤더 제목**: 30px, Bold, `#111827`
- **카드 제목**: 18px, SemiBold, `#111827`
- **본문**: 14px, Regular, `#374151`
- **캡션/라벨**: 12px, Regular, `#6b7280`
- **폰트**: System Font Stack (Apple System, Segoe UI, Roboto)

## 2. 헤더 영역

### 2.1 구성 요소
```
┌─────────────────────────────────────────────────────────────┐
│  QIP Trainer Dashboard        [한국어][English][Tiếng Việt] │
│                               [📄 PDF] [📤 파일 업로드]      │
└─────────────────────────────────────────────────────────────┘
```

### 2.2 기능 상세

#### 2.2.1 언어 선택기
- **위치**: 헤더 우측
- **구성**: 3개 언어 버튼 (한국어, English, Tiếng Việt)
- **스타일**: 
  - 비활성: 회색 배경 (`#f3f4f6`)
  - 활성: 흰색 배경 + 그림자
  - 호버: 밝은 회색 배경
- **동작**: 
  - 클릭 시 전체 UI 언어 즉시 변경
  - 로컬 스토리지에 선택 저장
  - 차트 라벨, 테이블 헤더, 버튼 텍스트 모두 변경

#### 2.2.2 PDF 다운로드 버튼
- **아이콘**: 문서 다운로드 SVG 아이콘
- **텍스트**: "PDF 다운로드"
- **동작**:
  1. 클릭 시 현재 보이는 대시보드 캡처
  2. jsPDF 라이브러리로 PDF 생성
  3. 파일명: `dashboard-report-YYYY-MM-DD.pdf`
  4. 포함 내용: 통계 카드, 활성 탭의 차트/테이블

#### 2.2.3 파일 업로드 버튼
- **아이콘**: 클라우드 업로드 SVG 아이콘
- **텍스트**: "파일 업로드"
- **동작**:
  1. 클릭 시 파일 선택 다이얼로그 오픈
  2. 허용 형식: `.xlsx`, `.xls`
  3. 파일 선택 후 자동 파싱 및 대시보드 업데이트

## 4. 통계 카드 섹션

### 4.1 레이아웃
```
┌─────────┬─────────┬─────────┬─────────┬─────────┐
│  총검증  │ 전체불량 │ 고위험  │  활동   │ 일평균  │
│  수량   │   률    │  TQC   │ 검사원  │  검증량  │
│ 35,160  │  2.5%   │  8명   │  25명   │  1,172  │
└─────────┴─────────┴─────────┴─────────┴─────────┘
```

### 4.2 각 카드 구성
1. **총 검증 수량**
   - 전체 Validation Qty 합계
   - 숫자 포맷: 천 단위 콤마
   - 색상: 기본 검정 (`#111827`)

2. **전체 불량률**
   - (총 Reject Qty / 총 Validation Qty) × 100
   - 소수점 2자리 표시
   - 색상 코딩:
     - < 3%: 녹색 (`#10b981`)
     - 3-5%: 주황색 (`#f59e0b`)
     - > 5%: 빨간색 (`#ef4444`)

3. **고위험 TQC**
   - 불량률 5% 이상인 TQC 수
   - "명" 단위 표시
   - 빨간색 강조 (`#ef4444`)

4. **활동 검사원**
   - 고유 Inspector ID 수
   - "명" 단위 표시

5. **일평균 검증량**
   - 총 검증 수량 / 작업일 수
   - 천 단위 콤마

### 4.3 애니메이션
- 데이터 로드 시: 0에서 실제 값으로 카운트업 (1초)
- 기간 변경 시: 페이드 전환 효과

## 6. 각 탭별 상세 구성

### 6.1 전체 현황 탭

#### 6.1.1 필터 섹션
```
구역: [전체 ▼] 
```
- 드롭다운으로 Building 선택
- 선택 시 모든 차트 실시간 업데이트

#### 6.1.2 차트 구성
```
┌─────────────────────┬─────────────────────┐
│   일별 불량률 추이   │   불량 유형 분포     │
│   (Line Chart)      │   (Doughnut Chart)   │
├─────────────────────┴─────────────────────┤
│          구역별 품질 현황                   │
│          (Bar Chart)                       │
└───────────────────────────────────────────┘
```

**일별 불량률 추이 (Line Chart)**
- X축: 날짜 (최근 30일)
- Y축: 불량률 (%)
- 색상: 빨간색 라인 (`#ef4444`)
- 배경: 연한 빨간색 그라데이션
- 호버: 툴팁으로 정확한 날짜와 값 표시
- 추세선: 점선으로 표시 (선택적)

**불량 유형 분포 (Doughnut Chart)**
- 중앙 홀: 40%
- 세그먼트: 최대 10개 유형
- 색상: 카테고리별 구분
- 범례: 우측 배치
- 호버: 해당 세그먼트 확대 + 툴팁

**구역별 품질 현황 (Bar Chart)**
- X축: Building 이름
- Y축: 불량률 (%)
- 색상 코딩:
  - < 3%: 녹색
  - 3-5%: 주황색
  - > 5%: 빨간색
- 호버: 정확한 수치 표시

### 6.2 TQC 분석 탭

#### 6.2.1 필터 섹션
```
구역: [전체 ▼]  불량률: [전체 ▼]
```

#### 6.2.2 TQC 테이블
```
┌────┬──────────┬────────┬──────┬────────┬────────┬────────┬──────────┬──────┐
│순위│ TQC ID   │  이름   │ 구역 │검증수량 │불량수량 │ 불량률 │주요불량  │ 상태 │
├────┼──────────┼────────┼──────┼────────┼────────┼────────┼──────────┼──────┤
│ 1  │ T001234  │ 홍길동  │ A동  │ 1,234  │  123   │ 9.97%  │ 스크래치 │ 🔴   │
└────┴──────────┴────────┴──────┴────────┴────────┴────────┴──────────┴──────┘
```

**테이블 특징**:
- 정렬: 불량률 기준 내림차순
- 상위 10명만 표시
- 클릭 가능한 행 (상세 모달 오픈)
- 상태 아이콘:
  - 🟢 양호 (< 3%)
  - 🟡 주의 (3-5%)
  - 🔴 위험 (> 5%)

#### 6.2.3 차트 영역
```
┌─────────────────────┬─────────────────────┐
│  TQC 불량률 분포     │   교육 우선순위      │
│  (Histogram)        │   (Bar Chart)        │
├─────────────────────┴─────────────────────┤
│      구역별 TQC Defect Missing 히트맵       │
│            (Heatmap Table)                 │
└───────────────────────────────────────────┘
```

**TQC 불량률 분포 (Histogram)**
- X축: 불량률 구간 (0-1%, 1-3%, 3-5%, 5-10%, 10%+)
- Y축: TQC 수
- 색상: 그라데이션 (녹색 → 빨간색)

**교육 우선순위 (Bar Chart)**
- 불량 유형 필터 제공
- 해당 불량이 높은 TQC Top 10
- 가로 막대 차트
- 클릭 시 TQC 상세 정보

**히트맵 테이블**
```
        Building A  Building B  Building C
TQC001     2.5%       3.2%       1.8%
TQC002     5.1%       2.3%       4.2%
```
- 색상 강도로 불량률 표현
- 셀 호버 시 정확한 수치

**히트맵**

- 색상: 발생 빈도 (연한 → 진한)
- 셀 크기: 고정
