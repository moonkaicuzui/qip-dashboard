## 🥇 추천 1위: PythonAnywhere (사실상 영구 무료)

### 왜 1위인가?
- ✅ **3개월마다 클릭 1번**이면 영원히 무료
- ✅ 로그인 기능, 데이터베이스 모두 지원
- ✅ 설정 가장 간단 (15분)
- ✅ HTTPS 자동 제공

### 🚀 빠른 시작 (15분)

#### 1. 계정 생성 (2분)
```
https://www.pythonanywhere.com
→ "Create a Beginner account" (무료)
→ 이메일 인증
```

#### 2. 파일 업로드 (5분)
PythonAnywhere 대시보드에서:
1. **Files** 탭 클릭
2. `/home/[username]/` 에서 **New directory**: `qip-dashboard`
3. 파일 업로드:
   - `webapp/app_simple.py`
   - `webapp/templates/` 폴더
   - `output_files/*.html` 파일들

#### 3. 웹 앱 생성 (3분)
1. **Web** 탭 클릭
2. **Add a new web app**
3. **Manual configuration** → **Python 3.9**

#### 4. WSGI 설정 (2분)
WSGI configuration file 편집:
```python
import sys
import os

username = 'YOUR_USERNAME'  # ← 여기만 변경!
project_home = f'/home/{username}/qip-dashboard'

sys.path.insert(0, project_home)
os.chdir(project_home)

from webapp.app_simple import app as application
```

#### 5. 완료! (1분)
1. **Reload** 버튼 클릭
2. 접속: `https://[username].pythonanywhere.com`

### 💡 3개월 갱신 자동화

크롬 확장 프로그램 사용:
1. "Auto Refresh Plus" 설치
2. PythonAnywhere 로그인 유지
3. 3개월마다 자동 갱신

## 🥈 추천 2위: Oracle Cloud (100% 영구 무료)

### Always Free 제공
- 🖥️ **VM 2개** (각 1GB RAM)
- 💾 **저장소 200GB**
- 🌐 **월 10TB 트래픽**
- ⚡ **갱신 불필요**

### 🚀 설정 방법 (30분)

#### 1. 계정 생성 (10분)
```
https://oracle.com/cloud/free/
→ Start for free
→ 신용카드 등록 (인증용, 과금 안됨)
```

#### 2. VM 생성 (5분)
```
Compute → Instances → Create Instance
- Shape: VM.Standard.E2.1.Micro (Always Free)
- OS: Ubuntu 22.04
- SSH 키 다운로드 (중요!)
```

#### 3. 서버 설정 (15분)
SSH 접속 후:
```bash
# 기본 패키지 설치
sudo apt update
sudo apt install -y python3-pip nginx

# Flask 앱 설치
pip3 install flask pandas numpy openpyxl

# 프로젝트 복사
scp -r webapp ubuntu@[IP주소]:~/

# 실행
python3 ~/webapp/app_simple.py
```

#### 4. 도메인 연결 (선택)
```
Public IP를 도메인에 연결
또는 IP로 직접 접속
```

## 🥉 추천 3위: GitHub Pages (정적 파일, 100% 무료)

### 특징
- 📁 HTML 파일 직접 호스팅
- 🔒 비밀번호 보호 불가 (공개)
- 🤖 GitHub Actions로 자동 업데이트

### 🚀 설정 방법 (10분)

#### 1. 저장소 생성
```bash
# GitHub에서 새 저장소 생성
git init
git add .
git commit -m "Initial"
git push origin main
```

#### 2. Pages 활성화
```
Settings → Pages
→ Source: Deploy from a branch
→ Branch: main, /docs
→ Save
```

#### 3. 파일 구조
```
docs/
├── index.html  (대시보드 HTML)
├── 2025_09.html
├── 2025_10.html
└── 2025_11.html
```

#### 4. 자동 배포
`.github/workflows/deploy.yml` 파일 추가 (위에서 생성함)

#### 5. 접속
```
https://[username].github.io/[repository]/
```
