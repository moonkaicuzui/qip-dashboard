# QIP Trainer Dashboard - 탭 및 팝업창 상세 문서

## 대시보드 개요
QIP Trainer Dashboard는 품질 검사 데이터를 시각화하고 분석하는 종합 대시보드입니다. 6개의 주요 탭으로 구성되어 있으며, 각 탭에는 다양한 차트, 테이블, 그리고 상세 정보를 제공하는 팝업창이 포함되어 있습니다.

---

## 📊 탭 1: 전체 현황 (Overview)

### 주요 구성 요소
1. **필터 섹션**
   - 구역(Building) 선택 필터
   - 전체 또는 특정 구역별 데이터 필터링

2. **일별 불량률 추이 차트**
   - Line Chart 형태
   - 최근 30일간 불량률 변화 표시
   - 추세선(Trend Line) 포함
   - 추세 지표(상승/하락/유지) 표시

3. **불량 유형 분포 차트**
   - Pie Chart 형태
   - 전체 불량의 유형별 비율 표시
   - 상위 10개 불량 유형 표시
   - 색상 구분된 레전드

4. **구역별 품질 현황 차트**
   - Bar Chart 형태
   - Building별 불량률 비교
   - 색상 코드: 
     - 녹색: 3% 미만
     - 노란색: 3-5%
     - 빨간색: 5% 초과

### 팝업창
이 탭에는 별도의 팝업창이 없습니다.

---

## 🔍 탭 2: TQC 분석 (TQC Analysis)

### 주요 구성 요소
1. **필터 섹션**
   - 구역(Building) 필터
   - 불량률 범위 필터 (0-1%, 1-3%, 3-5%, 5-10%, 10%+)

2. **TQC별 품질 성과 테이블**
   - 불량률 기준 상위 10명 표시
   - 컬럼: 순위, TQC ID, 이름, 구역, 검증 수량, 불량 수량, 불량률, 주요 불량 유형, 상태
   - 상태 배지: 정상(녹색), 주의(노란색), 긴급(빨간색)

3. **TQC 불량률 분포 히스토그램**
   - Bar Chart 형태
   - 불량률 구간별 TQC 인원 분포

4. **교육 우선순위 차트**
   - 불량 유형별 교육 대상자 표시
   - 드롭다운으로 불량 유형 선택 가능
   - 결함 누락률(Defect Missing Rate) 기준 정렬

5. **구역별 TQC Defect Missing 히트맵**
   - 각 구역별 TQC와 불량 유형의 교차 분석
   - 색상 강도로 누락률 표시

### 팝업창: TQC 상세 정보
**트리거**: TQC 테이블의 행 클릭 시

**팝업 내용**:
- **헤더 정보**
  - TQC 이름과 ID
  - 총 검증 수량
  - 불량 수량
  - 불량률
  - 작업 구역

- **4개 차트 그리드**
  1. **어딧터 검사 추이**: 일별 불량률 변화 및 추세선
  2. **구역별 불량률**: Building별 불량률 비교
  3. **모델별 불량률**: 상위 10개 모델의 불량률
  4. **TOP 3 Defect Missing**: 주요 3개 불량 유형 발생 건수

- **불량 유형 상세 리스트**
  - 2열 형태로 상위 10개 불량 유형과 발생 건수 표시

### 팝업창: 히트맵 상세 정보
**트리거**: 히트맵 셀 클릭 시

**팝업 내용**:
- TQC와 특정 불량 유형의 상세 정보
- 누락률(Missing Rate) 표시
- 총 검사 수량 대비 불량 건수
- 30일간 발생 추이 차트 및 추세선

## 📈 탭 3: 불량 분석 (Defects Analysis)

### 주요 구성 요소
1. **필터 섹션**
   - 구역(Building) 필터

2. **불량 유형별 발생 추이 차트**
   - Multi-line Chart
   - 주요 불량 유형의 일별 변화 표시

3. **불량 유형 상세 테이블**
   - 불량 유형 필터 포함
   - 컬럼: 불량 유형, 발생 건수, 비율, 주요 발생 구역

4. **구역별 불량 유형 히트맵**
   - Building과 불량 유형의 교차 분석
   - 색상 강도로 불량 강도 표시

5. **모델별 불량 현황 차트**
   - 모델 필터와 불량 유형 필터 포함
   - Bar Chart로 모델별 불량률 비교

6. **모델별 불량 유형 히트맵**
   - 모델과 불량 유형의 패턴 분석

### 팝업창: 불량 상세 정보
**트리거**: 히트맵 셀 클릭 시

**팝업 내용**:
- 구역/모델명
- 선택한 불량 유형
- 불량률 정보
- 관련 TQC 리스트 (불량 건수 포함)

## ⚠️ 탭 4: 위험도 분석 (Risk Analysis)

### 주요 구성 요소
1. **고위험 PO 카드**
   - 위험도가 높은 PO 번호 리스트
   - 각 PO의 불량률과 검증 수량 표시

2. **고위험 구역 카드**
   - 위험도가 높은 Building 리스트
   - 불량률과 관련 정보 표시

3. **고위험 라인 카드**
   - 위험도가 높은 생산 라인 리스트
   - 불량률과 검증 수량 표시

4. **위험도 예측 매트릭스**
   - Scatter Chart 형태
   - X축: 발생 빈도
   - Y축: 영향도
   - 버블 크기: 위험도

### 팝업창: 위험도 상세 정보
**트리거**: 고위험 항목 클릭 시

**팝업 내용**:
- 선택한 항목명 (PO/Building/Line)
- 총 검증 수량
- 총 불량 수량
- 불량률
- 관련 어딧터(Auditor) 목록
- 불량 유형별 발생 건수 리스트
- 관련 TQC 목록

## 📋 탭 5: 업무 지시 (Actions)

### 주요 구성 요소
1. **긴급 중점 관리 대상 카드**
   - 즉시 조치가 필요한 항목 리스트
   - 우선순위별 정렬
   - 각 항목의 위험 수준 표시

2. **어딧터별 추천 업무 카드**
   - 각 어딧터에게 할당된 권장 업무
   - 업무 유형과 대상 구역/라인 표시

3. **주간 액션 플랜**
   - 요일별 계획된 활동
   - 담당자와 목표 명시
   - 진행 상태 표시

### 팝업창
이 탭에는 별도의 팝업창이 없습니다.

## 🔄 탭 6: 지속성 (Sustainability) - **하이라이트**

### 주요 구성 요소

#### 1. **TQC 변동성 분석 카드**
- **필터**: 구역(Building) 선택
- **테이블 구성**:
  - TQC 이름
  - 평균 불량률
  - 변동 범위 (±%p)
  - 일관성 점수
  - 상태 배지 (정상/주의/긴급)
- 일관성 점수가 높은 순으로 상위 10명 표시

#### 2. **어딧터 활동 일관성 카드**
- 각 어딧터별 개별 카드 표시
- **카드 내용**:
  - 일평균 검증량
  - 일평균 TQC 수
  - 일평균 모델 수
  - 미니 차트: 일별 검증량 추이와 평균선

#### 3. **구역별 성과 일관성 차트**
- Bar Chart + 평균선
- 각 Building의 일관성 점수 비교
- 색상 코드:
  - 녹색: 25점 미만 (안정적)
  - 노란색: 25-50점 (보통)
  - 빨간색: 50점 초과 (불안정)
- **평가 가이드 텍스트**: 가장 변동성이 큰 구역에 대한 해석과 조치 제안

#### 4. **모델별 품질 일관성 차트**
- 상위 15개 모델의 일관성 점수
- Bar Chart + 전체 평균선
- 색상 코드는 구역별 차트와 동일
- **평가 가이드 텍스트**: 해석과 개선 제안

### 팝업창 1: TQC 일관성 상세
**트리거**: TQC 변동성 테이블의 행 클릭 시

**팝업 내용**:
- **상단 통계 (3개 컬럼)**:
  - 평균 불량률
  - 변동 범위
  - 일관성 점수

- **점수 산출 방법 설명**:
  - 계산 공식 표시
  - 가중치 설명 (평균 불량률 3% 미만 = 2.0, 7% 미만 = 1.5, 기타 = 1.0)
  - 낮은 점수가 높은 일관성을 의미함을 설명

- **일별 불량률 추이 차트**:
  - 실제 불량률 라인
  - 평균선 (주황색 점선)
  - 추세선 (회색 점선)
  - 추세 지표 (상승/하락/유지)

- **TOP 3 불량 유형 추세 차트**:
  - 주요 3개 불량 유형의 일별 발생 추이
  - 각 불량 유형별 추세선 포함

### 팝업창 2: 어딧터 일관성 상세
**트리거**: 어딧터 카드 클릭 시

**팝업 내용**:
- **어딧터 이름** (상단 중앙)

- **상단 통계 (3개 컬럼)**:
  - 일평균 검증량
  - 일평균 TQC 수
  - 일평균 모델 수

- **3개 차트 (1열 3행 배치)**:
  1. **일평균 검증 수량 차트**
     - 일별 검증량 라인
     - 추세선
     - 추세 지표
  
  2. **일평균 TQC 수 차트**
     - 일별 TQC 수 라인
     - 추세선
     - 추세 지표
  
  3. **일평균 모델 수 차트**
     - 일별 모델 수 라인
     - 추세선
     - 추세 지표

### 팝업창 3: 구역별 TQC 일관성 상세
**트리거**: 구역별 성과 일관성 차트의 막대 클릭 시

**팝업 내용**:
- **선택한 구역명과 설명 텍스트**

- **정렬 가능한 테이블**:
  - 컬럼 헤더 클릭으로 정렬 가능
  - 컬럼 구성:
    - TQC 이름 (25% 너비)
    - 총 검증 수량 (12%)
    - 평균 불량률 (12%)
    - 일관성 점수 (12%) - 기본 내림차순 정렬
    - 변동 범위 (12%)
    - 사분위 분포 시각화 (27%)

- **사분위 분포 시각화**:
  - 박스플롯 형태의 미니 차트
  - 파란 박스: Q1-Q3 범위
  - 빨간선: 중앙값(Median)
  - 녹색선: 평균(Average)

- **사분위 분포 읽는 방법 설명**:
  - 박스 왼쪽 끝: Q1 (하위 25%)
  - 박스 오른쪽 끝: Q3 (상위 25%)
  - 빨간선: 중앙값 (데이터의 중앙 값)
  - 녹색선: 평균 (산술 평균)
  - 중앙값과 평균의 차이가 데이터 치우침을 나타냄

### 팝업창 4: 모델 일관성 상세
**트리거**: 모델별 품질 일관성 차트의 막대 클릭 시

**팝업 내용**:
- **선택한 모델명**

- **불량 유형별 일관성 점수 테이블**:
  - 컬럼: 불량 유형, 평균 발생 건수, 발생 변동 범위, 일관성 점수
  - 일관성 점수가 높은 순으로 정렬
  - 변동성이 큰 불량 유형 식별 가능

## 🎯 지속성 탭의 핵심 가치

지속성 탭은 품질 관리의 **안정성과 예측 가능성**을 평가하는 핵심 도구입니다:

1. **TQC 성과 안정성**: 개별 검사원의 불량률 변동을 추적하여 교육이 필요한 대상 식별
2. **어딧터 활동 패턴**: 검사 활동의 일관성을 모니터링하여 업무 배분 최적화
3. **구역별 품질 변동성**: 특정 구역의 품질 불안정 요인 파악
4. **모델별 품질 패턴**: 제품 모델별 품질 일관성 분석으로 설계/공정 개선점 도출

각 팝업창은 더 깊은 수준의 분석을 제공하여, 관리자가 **데이터 기반의 의사결정**을 내릴 수 있도록 지원합니다.

## 시각적 디자인 요소

### 색상 체계
- **위험 수준**: 빨강(긴급) > 주황(주의) > 초록(정상)
- **차트 색상**: 파랑(주요), 회색(보조), 주황(평균/경고)
- **배경**: 연한 회색 계열로 계층 구분

### 인터랙션
- 호버 효과: 차트와 테이블 요소에 마우스 오버 시 하이라이트
- 클릭 가능 요소: 커서 변경 및 음영 효과
- 모달 애니메이션: 페이드인 효과

### 반응형 디자인
- 태블릿/모바일 대응
- 그리드 시스템 자동 조정
- 차트 크기 최적화
