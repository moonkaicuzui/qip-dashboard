## 🔧 수정된 주요 이슈

### 1. Bootstrap JavaScript 누락 (ROOT CAUSE)
- **문제**: Bootstrap CSS만 로드되고 JS가 누락되어 `ReferenceError: bootstrap is not defined` 발생
- **해결**: `bootstrap.bundle.min.js` CDN 추가
- **영향**: 모달이 완전히 작동하게 됨

### 2. 통화 기호 오류
- **문제**: 한국 원화(₩) 표시
- **해결**: 베트남 동(₫)으로 전체 교체
- **영향**: 베트남 공장 환경에 맞는 정확한 통화 표시

### 3. 이벤트 충돌
- **문제**: 인센티브 클릭 시 노드 접기/펴기 동시 발생
- **해결**: 이벤트 위임 및 stopImmediatePropagation 구현
- **영향**: 의도한 동작만 정확히 실행
