# 📱 대시보드를 모바일에서 보는 방법

## 🎯 목적
컴퓨터에 있는 인센티브 대시보드 HTML 파일을 모바일(스마트폰/태블릿)에서 볼 수 있게 합니다.

---

## 방법 1: 초간단 버전 (추천) ⭐

### 1단계: 서버 시작
터미널에서 다음 명령어 실행:
```bash
python3 간단한_모바일_뷰어.py
```

### 2단계: 모바일에서 접속
1. 모바일 브라우저(크롬, 사파리 등) 열기
2. 터미널에 표시된 주소 입력 (예: `http://192.168.1.100:8000`)
3. `output_files` 폴더 클릭
4. 원하는 대시보드 HTML 파일 클릭

---

## 방법 2: 편리한 버전 (QR 코드 포함)

### 1단계: 서버 시작
```bash
./start_mobile_server.sh
```
또는
```bash
python3 mobile_viewer.py
```

### 2단계: 모바일 접속
- QR 코드 스캔 또는
- 표시된 URL 직접 입력

### 특징:
- 대시보드 목록이 예쁘게 정리됨
- 클릭 한 번으로 대시보드 열기
- QR 코드로 편하게 접속

---

## ⚠️ 중요 사항

### ✅ 필수 조건
1. **컴퓨터와 모바일이 같은 와이파이에 연결되어 있어야 함**
2. 컴퓨터에서 Python이 설치되어 있어야 함

### 🚫 주의 사항
- 서버 실행 중에는 터미널 창을 닫으면 안 됨
- 종료하려면 `Ctrl+C` 누르기

### 🔧 문제 해결

**"연결할 수 없음" 오류가 나올 때:**
1. 컴퓨터와 모바일이 같은 와이파이인지 확인
2. 방화벽이 차단하고 있을 수 있음 → 허용 필요
3. IP 주소가 맞는지 확인

**QR 코드가 안 나올 때:**
```bash
pip install qrcode pillow
```
